About this site

This site is a place where I can write about the things I like to write about and showcase my lastest work. The site design is a custom one, created by me. It is built and published using Jekyll and turbo-charged with the Jekyll Assests plugin. The fonts used are Museo and Museo Sans by exljbris and the icon font generator is Fontello.
